# 🐉 Creature Art Mixer

## 🎶 Overview  
**Creature Art Mixer** is an **interactive audio-visual experience** that blends quirky creature imagery with dynamic sound manipulation powered by **Tone.js**. Users can trigger and modify sound loops by clicking on various creature parts and adjusting audio effects by dragging elements like the Cloud, Meteor, and UFO.

---

## 🚀 Features  
- 🎵 **Interactive Creature Elements**: Click on different creature parts (nose, ears, mouth, etc.) to toggle sound loops.  
- 🎚 **Real-Time Audio Effects**:  
  - 🎤 Drag the **Cloud** to control **Reverb**.  
  - 🎸 Drag the **Meteor** to control **Distortion**.  
  - 🎷 Drag the **UFO** to control **Delay**.  
- 🌈 **Dynamic Visual Feedback**: Active elements pulsate with a **rainbow glow effect**.  
- 🔁 **Custom Sound Loops**: Preloaded audio loops combine to create a unique mix.  
- 📌 **Neumorphic Navigation UI**: Seamless access to **Food4Thoth** projects and related content.

---

## 🛠 Technologies Used  
- **HTML, CSS, JavaScript**  
- **[Tone.js](https://tonejs.github.io/)** for Web Audio API interactions  
- **Neumorphic UI Design** for an immersive visual experience
