# FOOD4THOTH Music Library

## Overview

The FOOD4THOTH Music Library is a curated digital music collection designed to showcase a diverse range of sounds, featuring multiple progressive rock albums from INPROGRESSION and an experimental solo piece by De’Jahn. This library utilizes Vue.js for dynamic interactivity, allowing users to explore and listen to tracks seamlessly.

This project is an adaptation and enhancement of an original Vue-based music player, with additional customization by De’Jahn and FOOD4THOTH, expanding its functionality and aesthetics.

---

#  Features

## 🎵 Music Player Functionality

•	Browse a growing catalog of progressive rock, experimental music, and more.
•	Filter music by album for easy navigation.
•	Toggle song details (artist, album, release year, genre).
•	Seamless playback with dynamic highlighting of currently playing tracks.
	
---

## 🌌 Custom Design & UX Enhancements

•	Dark-themed interface with a neumorphic navigation menu for an immersive experience.
•	Interactive elements with smooth transitions and hover effects.
•	Background visuals from Unsplash for an atmospheric aesthetic.
	
---

## 🔧 Technical Features

•	Built with Vue.js for real-time interactivity.
•	Modular CSS design with neumorphic UI elements.
•	Dynamic audio playback that ensures only one song plays at a time.
•	Expandable navigation system with submenu categories.
	
---

# Music Collection

Albums from INPROGRESSION

•	Down The Rabbit Hole (2008 A.D.)
•	Time is Breaking (2009 A.D.)
•	Burned (2007 A.D.)

Solo Track

•	Funkin’ Around (2012 A.D.) by De’Jahn – A 40-minute experimental sonic journey.
	
---

# How to Use
1.	Navigate the Library

	•	Click on the ☰ Nav button to access different sections.
	
	•	Expand submenus for related projects and resources.
	
2.	Play Music

	•	Click on a track title to view its details.
	
	•	Press the play button to start playback.
	
	•	Only one track will play at a time.

3.	Filter by Album

	•	Use the dropdown menu to filter tracks by album.
	
4.	Explore More Content

	•	The navigation menu links to additional projects such as Community Gardens, Anarchy Comics, Sound Visualizers, Tarot, I Ching, and more.
