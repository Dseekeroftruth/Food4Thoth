    - Seamless sprite animation of 9 illustrated frames.

# How It Works

### Sprite Animation

The animation simulates walking using a sprite sheet broken into separate frames (Untitled.png through Untitled9.png). Each frame is cycled via CSS or JavaScript to achieve smooth motion.

### Background Cycling

In the full version, backgrounds transition with each animation loop:
- BG2.webp
- BG4.webp
- BG9.webp
- BG10.webp

This gives the illusion of traversing shifting dimensions.

---

# Live Controls & Integration

The preview supports external control and theme toggling via window.postMessage:

window.postMessage({ triggerAnimation: true });
window.postMessage({ darkMode: true });

- triggerAnimation: Starts sprite animation.
- darkMode: Toggles dark mode styling (for transparent version).
